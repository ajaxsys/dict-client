Usage

Try it: Simply select any word on this web page by your mouse,then see what happen?

Use on other web pages: Simply drag this link DICT to your bookmark(It calls bookmarklet). Then click this bookmarklet on any web pages.

What can it do:

  • Search selection keyword from any pages
  • Get text from link/image/select_box, when holding mouse on target text.
  • Support Chrome/Safari/Firefox
  • Support HTTP page and almost all HTTPS pages
  • Support PDF files.(Read this post for detail)
