#dict-client
================================

##DEMO

http://dict-admin.appspot.com/

##Run local:

Clone `dict-client` to local

###Compile
    cd dict-client
    npm install
    grunt
    grunt watch

###Run simple http server
    cd tool
    node run-server-js

Then you can visit URL:
 http://localhost:8443/build/test/test.html
