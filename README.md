http://ajaxsys.github.io/dict-client/dict_en.html

##Run local:

Clone `dict-client` to local

###Compile
    cd dict-client
    npm install
    npm install -g grunt
    grunt

###Watch source change
    grunt watch

###Run simple http server
    cd tool
    run-server-js

Now you can visit URL:
 http://localhost:8443/target/build/test/test.html
